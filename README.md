# React Snake Game

Archived React project implementing a classic Snake game for the browser.

## Overview

This project is a browser-based Snake game built with React. The player controls the snake, collects food, grows in length, and the game speed increases as the score progresses.

The project is kept public as part of my early frontend development background.

## Live Demo

Demo: https://itkrivoshei.github.io/React-Snake/

If the repository has been renamed, the GitHub Pages URL may have changed. Use the active demo link from the repository homepage.

## Features

- Classic Snake gameplay
- Food spawning logic
- Snake growth after eating food
- Increasing game speed
- Keyboard-based controls
- Component-based React structure
- GitHub Pages deployment
